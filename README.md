# ZeroSkid

ZeroSkid is a project focused on using AI to detect wet road conditions early, with the goal of helping prevent accidents before they happen.

## Repository Structure

- docs/ — contains the website code (HTML, CSS, etc.)
- images/ — final outputs like accuracy graphs, loss graphs, and confusion matrices
- roads/ — dataset used for training and testing (wet vs. dry road images)

## Results

- Best validation accuracy is around 75–80%
- Lower learning rates performed more consistently across runs
- Model used: AlexNet (CNN)

## Purpose

This project shows how AI can be applied to a real-world safety problem. The focus is on using computer vision to identify road conditions before they become dangerous for drivers.
